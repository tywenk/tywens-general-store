# Architecture Scout

Map, analyze, and plan migrations for legacy codebases.

## Install

```sh
/plugin marketplace add tywenk/tywens-general-store
/plugin install architecture-scout@tywens-general-store
```

### Local install

If you've cloned this repo, you can install from the local path instead:

```sh
/plugin marketplace add ./plugins
/plugin install architecture-scout
```

## Usage

```
/architecture-scout:scout
```
